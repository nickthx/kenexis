This is a NEW project scaffold. The existing spsjoa-website codebase (Next.js 14, Tailwind v3) is the prior art but this phase creates a fresh Next.js 15 project. The research document (01-RESEARCH.md) contains all verified patterns, exact commands, and code examples needed.

Key constraints from research:
- Tailwind v4: Use @theme directive in CSS, NOT tailwind.config.ts
- Use @import "tailwindcss" NOT @tailwind directives
- Use tw-animate-css NOT tailwindcss-animate
- Use tailwind-merge v3.x NOT v1.x
- shadcn style: "new-york" (default is deprecated)
- components.json tailwind.config must be blank string for v4
- React 19: No forwardRef needed
- Navy #0a1628 is very dark -- map to 900 position in shade scale
- Orange #e87722 text on white fails WCAG AA -- use orange-700+ for text on light backgrounds
</context>
