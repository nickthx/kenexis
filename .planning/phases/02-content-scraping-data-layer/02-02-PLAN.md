**Unsplash direct download pattern:**
Unsplash images can be downloaded directly via their photo URLs. Use the format:
`https://images.unsplash.com/photo-{ID}?w=1920&q=80&auto=format`

The `w=1920` parameter gets a 1920px-wide image suitable for hero sections. The `q=80` keeps quality high while managing file size.

**Search and download process:**
1. Use `curl` to search Unsplash pages for relevant industrial images, OR use known Unsplash photo IDs for industrial/refinery/chemical plant imagery
2. For each image, download at 1920px width

**Required images and search terms:**

| Target File | Search Terms | Usage |
|-------------|-------------|-------|
| hero-refinery.jpg | oil refinery, petrochemical plant, refinery at dusk | Homepage hero background |
| hero-control-room.jpg | industrial control room, SCADA panel | Software pages hero |
| hero-safety-equipment.jpg | industrial safety, PPE, hard hat worker | Services pages hero |
| hero-industrial-plant.jpg | chemical plant, industrial facility | About/company hero |
| bg-industrial-1.jpg | industrial pipes, refinery piping | Section background accent |
| bg-industrial-2.jpg | industrial valve, gas pipeline | Section background accent |
| services-hero.jpg | process safety, industrial engineer | Services landing page |
| software-hero.jpg | computer dashboard, software interface, tech | Software landing page |
| training-hero.jpg | professional training, classroom, learning | Training page hero |
| about-hero.jpg | engineering team, corporate office | About page hero |

**Download approach:**
1. Use curl to fetch Unsplash search results pages and identify specific photo IDs, OR use WebFetch to browse Unsplash search pages
2. Download each image using `curl -L -o public/images/stock/{filename}.jpg "https://images.unsplash.com/photo-{ID}?w=1920&q=80&auto=format"`
3. If Unsplash rate-limits or blocks, try Pexels as alternative: `https://images.pexels.com/photos/{ID}/pexels-photo-{ID}.jpeg?auto=compress&w=1920`

**Fallback if automated download is blocked:**
If both Unsplash and Pexels block automated downloads, create high-quality placeholder gradient images using a Node.js script with the Canvas API or simply create CSS gradient placeholder files. Note this in the summary for future manual replacement.

**Quality checks:**
- Each image should be at least 100KB (real photo, not a tiny thumbnail)
- Each image should be 1920px wide or close to it
- File format: JPEG (not PNG -- photos compress better as JPEG)

**License:** All Unsplash and Pexels images are free for commercial use with no attribution required (though attribution is appreciated).
