    **Step 3: Create `src/hooks/use-scroll-header.ts`:**
    A "use client" hook that tracks whether the user has scrolled past a threshold (default 50px). Uses passive scroll event listener for performance. Returns a boolean `scrolled` state. Pattern from research:
    ```typescript
    "use client";
    import { useState, useEffect } from "react";
    export function useScrollHeader(threshold = 50) {
      const [scrolled, setScrolled] = useState(false);
      useEffect(() => {
        const handleScroll = () => setScrolled(window.scrollY > threshold);
        handleScroll();
        window.addEventListener("scroll", handleScroll, { passive: true });
        return () => window.removeEventListener("scroll", handleScroll);
      }, [threshold]);
      return scrolled;
    }
    ```

    Create `src/components/layout/breadcrumbs.tsx` as a "use client" component.

    Implementation:
    - Import `usePathname` from `next/navigation`.
    - Import shadcn `Breadcrumb`, `BreadcrumbItem`, `BreadcrumbLink`, `BreadcrumbList`, `BreadcrumbPage`, `BreadcrumbSeparator` from `@/components/ui/breadcrumb`.
    - Import `buildBreadcrumbMap` from `@/lib/navigation-utils`.
    - Import `Link` from `next/link`.
    - Import `Fragment` from `react`.

    Logic:
    1. Get `pathname` from `usePathname()`.
    2. If `pathname === "/"`, return `null` (no breadcrumbs on homepage per research pitfall #4).
    3. Call `buildBreadcrumbMap()` to get the label lookup.
    4. Split pathname into segments: `pathname.split("/").filter(Boolean)`.
    5. Build cumulative crumbs array: for each segment index `i`, create `{ href: "/" + segments.slice(0, i+1).join("/"), label: map[href] || capitalize(segment) }`.
    6. Render:
       ```
       <Breadcrumb className="px-4 py-3 max-w-7xl mx-auto">
         <BreadcrumbList>
           <BreadcrumbItem>
             <BreadcrumbLink asChild><Link href="/">Home</Link></BreadcrumbLink>
           </BreadcrumbItem>
           {crumbs.map((crumb, i) => (
             <Fragment key={crumb.href}>
               <BreadcrumbSeparator />
               <BreadcrumbItem>
                 {i === crumbs.length - 1
                   ? <BreadcrumbPage>{crumb.label}</BreadcrumbPage>
                   : <BreadcrumbLink asChild><Link href={crumb.href}>{crumb.label}</Link></BreadcrumbLink>
                 }
               </BreadcrumbItem>
             </Fragment>
           ))}
         </BreadcrumbList>
       </Breadcrumb>
       ```

    Also add a helper `capitalize` function (local, not exported): takes a string, replaces hyphens with spaces, capitalizes first letter of each word. This is the fallback when a path segment is not found in the breadcrumb map.
